üîç Dataset Insights

Target Distribution

  • Not Cancelled: 19,870 bookings (67.36%)
  • Cancelled: 9,630 bookings (32.64%)
  • Imbalance Ratio: 2.06:1
  • Dataset is imbalanced - addressed using class weights and scale_pos_weight in models

Key Features Analysis

  • Lead Time: Higher lead time strongly correlates with cancellations (Median: Cancelled 88 days vs Not Cancelled 42 days)
  • Special Requests: More special requests indicate lower cancellation (shows customer commitment)
  • Price: Slight correlation with cancellation behavior
  • Repeat Customers: Have lower cancellation rates compared to first-time guests
  • Meal Type & Room Type: Show distinct cancellation patterns across categories

Techniques Used

  • Stratified K-Fold Cross-Validation: 7 folds for robust evaluation
  • Imbalance Handling: Class weights and scale_pos_weight parameters
  • Feature Selection: Mutual Information-based selection of top 50 features
  • Ensemble Methods: Weighted averaging and soft voting
  • Hyperparameter Tuning: Optimized learning rate, tree depth, regularization
  • Calibration: Isotonic regression for probability calibration
